# Dispel Game File Extractor

## Overview

This is a technical research tool for analyzing the file formats used by the Dispel game. The project focuses on reverse engineering file structures, data formats, and technical specifications for educational and interoperability purposes.

The project analyzes and documents:

- **File formats**: Binary and text-based game data structures

- **Inter-file relationships**: How different file types interact
